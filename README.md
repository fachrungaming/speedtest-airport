# AirportTech Speedtest App

A futuristic, high-tech Speedtest application designed with an immersive "Airport Infrastructure" theme. Optimized for network audits and digital evidence collection.

## Features
- **Futuristic UI**: Glassmorphism, neon accents, and high-tech animations.
- **Boot Sequence**: Simulated system boot sequence for an immersive experience.
- **Live Console**: Real-time technical log stream.
- **Overlay Camera**: Capture network audits with a camera overlay containing real-time data.
- **Advanced Evidence Capture**: Downloads high-quality JPEGs with full HUD data (Mbps, Ping, Time, GPS, Location).
- **Library System**: View, open, and manage (delete) captured evidence.
- **Camera Switching**: Toggle between front and back cameras for mobile use.

## Technologies Used
- HTML5 / CSS3 (Tailwind CSS via CDN)
- Vanilla JavaScript
- Font: Space Grotesk & JetBrains Mono

## How to use
1. Open `menu utama.html` in your browser.
2. Click "GO" to run the speedtest.
3. Toggle "Overlay Kamera" to enter the evidence capture mode.
4. Use the "Capture" button to save an audit snapshot.

## Developed for
Sultan Hasanuddin International Airport (UPG) Network Audits.
